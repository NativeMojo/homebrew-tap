# NativeMojo Homebrew Tap

Homebrew tap for [Mojo Pulse](https://github.com/NativeMojo/mojo-pulse) and other NativeMojo macOS apps.

## Install

```sh
brew install --cask NativeMojo/tap/mojo-pulse
```

Or tap first, then install:

```sh
brew tap NativeMojo/tap
brew install --cask mojo-pulse
```

Mojo Pulse is Developer ID–signed and notarized by Apple, and auto-updates itself via Sparkle.
